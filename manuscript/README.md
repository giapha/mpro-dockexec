# Manuscript

**Title:** *Measuring the re-executability of a computational claim: a deterministic reporting standard and a noise-calibrated re-execution protocol for molecular docking*

**Authors:** Vincent He (corresponding, Vincentgiap@gmail.com), Eric Wang, Cris Nguyen — NewScience Lab.

## Abstract

A molecular docking score is the output of a workflow, not a standalone fact: it depends on the receptor, the ligand, the software, the search box, the random seed, and several preparation choices. Whether a published score can be re-run is therefore a measurable property of its reporting record, yet the field has had no operational way to measure it. We introduce that measurement layer — three coupled parts — and field-test it on the open-access SARS-CoV-2 main-protease (Mpro) docking literature. First, MERS-Dock, a 16-field Minimum Executable Reporting Set tiered by execution role; unlike documentation standards in adjacent fields (FAIR, ARRIVE, MIAME, datasheets, model cards), it maps reported fields to independent re-executability. Second, an E0–E4 executability ladder assigned by a deterministic decision rule over source-span-verified field states, turning "re-executable" into a computable, reviewer-validated label. Third, a re-execution protocol that calibrates its reproduction tolerance to the docking engine's own run-to-run noise floor rather than an arbitrary threshold. Every field state is anchored to a verbatim span and confirmed by a deterministic substring check, so the audit is a traceable measurement, not an opaque model output. Applied to 236 open-access Mpro papers, the method resolves a clear gradient: 8.1% are directly re-executable (E3; 95% CI 5.2–12.2), 44.1% executable only after explicit assumptions (E2), 47.9% execution-blocked by a missing foundational field (E1), and none fully reproducible with a disclosed seed and configuration (E4; 0/236); mean field completeness is 49.1%, and it localizes one dominant fixable gap — the search-box centre, present in only 33.9% of papers (absent from two-thirds). The label is reviewer-grounded: two independent human reviewers agree at Cohen's κ = 0.926, reproduced at the same value by the deterministic classifier and a blind language-model rater (holistic first-pass 0.075). As a bounded proof of concept, we re-docked 48 quality-controlled claims across three Mpro targets with AutoDock Vina 1.2.7 (box centres span-parsed for all 48; box sizes recoverable for only 12, the other 36 defaulted to a 25 Å cube). Against the engine's measured self-noise (median 0.04 kcal/mol), the proximal driver of reproduction was box availability: claims run with a reported box size reproduced to a median |Δ| of 0.33 kcal/mol versus 0.91 for those forced onto the default — a significant ~2.7-fold gap (Mann–Whitney p = 0.024). The executability class co-varies but is confounded with box-size source and underpowered (E3 n = 6, largely one paper; p = 0.07), so we read the label not as a reproducibility predictor but as an executability gate — which claims can be cleanly re-executed at all. Because the corpus is open-access and text-extractable, these figures are an upper bound on the paywalled literature, and the E-class describes the reporting record, not the correctness of the science. We release Mpro-DockExec — the standard, the deterministic classifier, and the re-execution scripts — as a reusable measurement layer that tells a reader, or an automated agent, what is checkable in a published computational claim; it validates labels rather than re-running the literature at scale.
